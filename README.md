# projeto_restaurante-8-teste

A ideia inicial é projetar uma aplicação Web que mecanize as tarefas sem complicar procedimentos simples, como anotar um pedido ou entregar um produto em sua devida mesa. Além disso, o sistema não poderá mudar a rotina da empresa e nem dos funcionários que farão uso do software, como também o protótipo deve auxiliar a gerencia do bar e restaurante.
